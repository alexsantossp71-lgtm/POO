O Que é Java?

Java é uma linguagem de programação Orientada a Objetos, fortemente tipada e híbrida (compilada e interpretada). Seu lema é "Write Once, Run Anywhere" (WORA), ou seja, escreva uma vez, rode em qualquer lugar.

Como Funciona?

Diferente de C++ que compila para código de máquina nativo, o compilador Java (javac) transforma seu código fonte (.java) em um código intermediário chamado Bytecode (.class).

Quem executa esse Bytecode é a JVM (Java Virtual Machine). É ela que garante que o programa rode igual no Windows, Linux ou Mac.

Estrutura Básica

Todo programa Java começa pelo método main. Em Java, tudo deve estar dentro de uma classe. Todas as classes herdam implicitamente da classe Object. 


// Nome do arquivo deve ser OlaMundo.java
public class OlaMundo {
    // Ponto de entrada da aplicação
    public static void main(String[] args) {
        System.out.println("Olá, Java!");
    }
}
