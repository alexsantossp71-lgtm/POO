Coleções

Estruturas de dados poderosas.

O que são?

O Java Collections Framework é uma arquitetura unificada para representar e manipular coleções (grupos de objetos). Ele fornece estruturas de dados prontas como Listas, Conjuntos (Sets) e Mapas, além de algoritmos para processá-los (ordenação, busca, etc.).

Exemplo em Java

import java.util.*;

public class ExemploColecoes {
    public static void main(String[] args) {
        // LIST: Ordenada, permite duplicatas
        List frutas = new ArrayList<>();
        frutas.add("Maçã");
        frutas.add("Banana");
        frutas.add("Maçã"); // Permitido
        
        System.out.println("Lista: " + frutas);

        // SET: Não ordenada, NÃO permite duplicatas
        Set numeros = new HashSet<>();
        numeros.add(1);
        numeros.add(2);
        numeros.add(1); // Ignorado
        
        System.out.println("Set: " + numeros);

        // MAP: Pares Chave-Valor
        Map idades = new HashMap<>();
        idades.put("Ana", 25);
        idades.put("Carlos", 30);
        
        System.out.println("Idade da Ana: " + idades.get("Ana"));
    }
}

Principais Interfaces

  • List: Sequência ordenada (ex: ArrayList, LinkedList).
  • Set: Coleção sem duplicatas (ex: HashSet, TreeSet).
  • Map: Mapeamento de chave para valor (ex: HashMap, TreeMap).
  • Queue: Fila para processamento (ex: PriorityQueue).
