Streams e Serialização

Processamento funcional e persistência.

Streams API

A Streams API (Java 8+) permite processar sequências de elementos de forma declarativa e funcional. É excelente para operações em coleções, como filtrar, mapear e reduzir dados.

List nomes = Arrays.asList("Ana", "Beto", "Carlos", "Bia");

// Filtrar nomes que começam com 'B', transformar em maiúsculas e imprimir
nomes.stream()
     .filter(n -> n.startsWith("B"))
     .map(String::toUpperCase)
     .forEach(System.out::println);
// Saída: BETO, BIA

Serialização

Serialização é o processo de converter o estado de um objeto em um fluxo de bytes, para que possa ser salvo em um arquivo, banco de dados ou enviado pela rede. A Desserialização é o processo inverso.

import java.io.*;

// A classe deve implementar Serializable
class Pessoa implements Serializable {
    String nome;
}

// Salvando o objeto (Serialização)
ObjectOutputStream out = new ObjectOutputStream(new FileOutputStream("pessoa.dat"));
out.writeObject(new Pessoa());
out.close();
