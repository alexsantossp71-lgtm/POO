Redes

Conectando aplicações distribuídas.

O que é?

A programação de redes em Java permite criar aplicações que se comunicam através de uma rede (como a Internet ou uma rede local). O pacote java.net fornece classes poderosas para trabalhar com protocolos de baixo nível (TCP, UDP) e alto nível (HTTP).

Exemplo: Cliente-Servidor Simples (TCP)

// --- SERVIDOR ---
import java.net.*;
import java.io.*;

public class Servidor {
    public static void main(String[] args) throws IOException {
        ServerSocket server = new ServerSocket(12345);
        System.out.println("Servidor ouvindo na porta 12345...");
        
        // Bloqueia até um cliente conectar
        Socket cliente = server.accept();
        System.out.println("Cliente conectado: " + cliente.getInetAddress().getHostAddress());
        
        server.close();
    }
}

// --- CLIENTE ---
import java.net.*;
import java.io.*;

public class Cliente {
    public static void main(String[] args) throws IOException {
        // Tenta conectar ao servidor no localhost
        Socket socket = new Socket("127.0.0.1", 12345);
        System.out.println("Conectado ao servidor!");
        
        socket.close();
    }
}

Conceitos Chave

  • IP: Endereço que identifica um computador na rede.
  • Porta: Identifica um processo/aplicação específica no computador.
  • Socket: Ponto final de uma conexão de comunicação bidirecional.
  • TCP: Protocolo confiável (garante entrega e ordem).
  • UDP: Protocolo rápido, mas não confiável (streaming, jogos).
