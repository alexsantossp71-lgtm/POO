Threads

Programação Concorrente e Paralela.

O que são?

Uma Thread é a menor unidade de processamento que pode ser agendada por um sistema operacional. Em Java, o uso de Threads permite que seu programa execute múltiplas tarefas "ao mesmo tempo" (concorrência), aproveitando melhor os processadores modernos com múltiplos núcleos.

Exemplo em Java

// Opção 1: Estender a classe Thread
class MinhaThread extends Thread {
    public void run() {
        System.out.println("Thread rodando em paralelo!");
    }
}

// Opção 2: Implementar Runnable (Melhor prática)
class MinhaTarefa implements Runnable {
    public void run() {
        for(int i=0; i<5; i++) {
            System.out.println("Tarefa executando: " + i);
        }
    }
}

public class Main {
    public static void main(String[] args) {
        MinhaThread t1 = new MinhaThread();
        t1.start(); // Inicia a thread

        Thread t2 = new Thread(new MinhaTarefa());
        t2.start(); // Inicia a segunda thread
        
        System.out.println("Método main continua executando...");
    }
}

Conceitos Chave

  • start(): Método que inicia a execução da thread (chama o run).
  • run(): Onde fica o código que a thread vai executar.
  • sleep(): Pausa a thread por um tempo determinado.
  • synchronized: Evita que duas threads acessem o mesmo recurso ao mesmo tempo, prevenindo inconsistências.
