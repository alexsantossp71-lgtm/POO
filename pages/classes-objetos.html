Classes e Objetos

A base da Programação Orientada a Objetos.

O que são?

Classes são como "moldes" ou "plantas" que definem a estrutura e o comportamento de objetos. Elas especificam quais atributos (dados) e métodos (ações) os objetos criados a partir dela terão.

Objetos são instâncias dessas classes. Se a classe é a planta de uma casa, o objeto é a casa construída em si. Você pode criar múltiplos objetos a partir de uma única classe, cada um com seus próprios valores para os atributos.

Exemplo em Java

// Definição da Classe
public class Carro {
    // Atributos (Características)
    String modelo;
    String cor;
    int ano;

    // Métodos (Comportamentos)
    void acelerar() {
        System.out.println("O " + modelo + " está acelerando: Vrum vrum!");
    }
    
    void frear() {
        System.out.println("O " + modelo + " parou.");
    }
}

// Classe Principal para testar
public class Main {
    public static void main(String[] args) {
        // Instanciando objetos (Criando carros)
        Carro carro1 = new Carro();
        carro1.modelo = "Fusca";
        carro1.cor = "Azul";
        carro1.ano = 1975;

        Carro carro2 = new Carro();
        carro2.modelo = "Ferrari";
        carro2.cor = "Vermelho";
        carro2.ano = 2023;

        // Usando os objetos
        carro1.acelerar(); // Saída: O Fusca está acelerando: Vrum vrum!
        carro2.acelerar(); // Saída: O Ferrari está acelerando: Vrum vrum!
    }
}

Pontos Chave

  • Classe: Definição abstrata (molde).
  • Objeto: Instância concreta da classe.
  • Atributos: Variáveis que guardam o estado do objeto.
  • Métodos: Funções que definem o comportamento do objeto.
  • new: Palavra-chave usada para criar um novo objeto.
